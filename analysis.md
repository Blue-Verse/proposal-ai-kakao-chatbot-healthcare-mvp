- 가중 평균 50% 절감 → 68 × 0.5 = **34 M/D**
- 외부 API 연동 + 본인인증 → 버퍼 +15%
- 조정 후: 34 × 1.15 ≈ **39 M/D** (보수적 산정 시 36~40 M/D 범위)

### 달력 일수 변환

- 36~40 M/D × (7/5 주말 보정) = **약 50~56일**
- 클라이언트 예상 60일 vs 산정 50~56일 → 산정이 더 짧음
- 판정: **클라이언트 60일 그대로 사용**. 1순위 일정 준수에 부합, 단계별 마일스톤으로 완성도 확보

### 6.4 클라이언트 질문 답변

**Q1. 초기 스타트업 MVP 또는 외부 API 연동 기반 서비스 개발 경험이 있으신가요?**

예. 헬스케어 SaaS(시니어 주간보호 관리 플랫폼)에서 OpenAI 기반 건강 분석·알림 시스템을 6 플랫폼 구조로 MVP부터 정식 운영까지 직접 수행했습니다. 또한 핀테크 VC 펀드 관리 플랫폼에서 NICE KYC 본인인증, ChatGPT, 다중 규제 API 등 15+ 외부 API를 단일 백엔드에 안정 연동한 경험이 있습니다. 외부 API 장애 대응(재시도·캐싱·폴백)을 표준 패턴으로 운영하고 있습니다.

**Q2. 카카오톡 채널 기반 챗봇 서비스를 실제로 개발 및 운영해본 경험이 있으신가요?**

카카오톡 채널 직접 운영 경험은 별도 포트폴리오로 안내드릴 수 있으며, 동등 수준의 챗봇·AI 에이전트 설계 경험(AI Agent 프레임워크 — 134+ 스킬 모듈, 멀티 LLM 오케스트레이션, 프롬프트·컨텍스트 관리, 외부 도구 자동 호출)을 다수 보유하고 있습니다. 카카오 i 오픈빌더 스킬서버 방식과 Webhook 직접 호출 방식의 트레이드오프(5초 응답 제한, Callback API 활용, 멀티턴 상태관리)는 사전 설계 완료 가능합니다.

**Q3. OpenAI(GPT) 또는 유사 AI API를 활용한 실제 서비스 구축 경험이 있으신가요?**

예. 1) 헬스케어 SaaS(Harmony Link)에서 OpenAI 기반 건강 데이터 분석·보호자 리포트 자동 생성, 2) 핀테크 SaaS(Series-B)에서 ChatGPT 기반 AI 투자 보고서 자동 생성(50+ 페이지 운영), 3) AI Agent 프레임워크에서 멀티 LLM 오케스트레이션·프롬프트 엔지니어링·토큰 비용 가드를 직접 설계·운영하고 있습니다.

**Q4. 공공기관 API 연동 경험이 있으신가요?**

공공데이터포털(data.go.kr) 계열 및 정부·지자체 API 연동 경험이 있으며, 응답 정규화, 호출 제한 대응, 캐싱·폴백 등 안정 연동 패턴을 표준화하여 적용합니다. 공고에 명시된 3~5개 공공 API는 NDA 후 명세 공유 시 단계별 연동 일정으로 매핑하여 제공드리겠습니다.

**Q5. 개발 완료 후 월 단위 유지보수를 맡아주실 수 있나요? (유지보수 비용 포함 견적 가능 여부)**

예. MVP 출시 이후 월 단위 유지보수 및 2차/3차 고도화 연속 협업을 적극 제안드립니다. 1개월 무상 하자보수는 본 견적에 포함되어 있으며, 그 이후 월 단위 유지보수는 운영 범위(이슈 대응 SLA, 정기 점검, 기능 추가 일정)에 따라 80~150만원/월 범위에서 별도 견적으로 제안드릴 수 있습니다. 미팅에서 운영 범위 협의 후 확정하겠습니다.
